# Set-SnipeitAccessoryOwner

## SYNOPSIS
Checkout accessory

## SYNTAX

```
Set-SnipeitAccessoryOwner [-id] <Int32[]> [-assigned_to] <Int32> [[-note] <String>] [[-url] <String>]
 [[-apiKey] <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Checkout accessory to user

## EXAMPLES

### EXAMPLE 1
```
Set-SnipeitAccessoryOwner -id 1 -assigned_to 1  -note "testing check out to user"
```
